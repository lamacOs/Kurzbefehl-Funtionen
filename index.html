lamacOs Kurzbefehle Funktionen

Anleitung

1. Kopieren Sie den Link der gewünschten Funktion.
2. Öffnen Sie die Kurzbefehle App.
3. Fügen Sie die Aktion „Inhalte von URL abrufen“ hinzu.
4. Fügen Sie den kopierten Link ein.
5. Falls Eingabe nötig ist, fügen Sie eine Variable am Ende hinzu (z. B. „Nach Eingabe fragen“).
